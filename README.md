# 概要
このリポジトリは、CLI操作に慣れるために自作したコマンドラインツール filetoolのコードをまとめたものです。 filetoolはテキストファイルの操作・確認を簡単に行えるCLIツールです。

# 実行環境
- Ruby 3.2.0  
- macOS

# 内容
filetoolは、テキストファイルを操作・確認するためのCLIツールです。
以下のコマンドを備えています。

- showコマンド : ファイルの内容を表示
- statsコマンド : 行数・単語数・文字数を表示
- addコマンド : テキストをファイル末尾に追記
- searchコマンド : 文字列を検索し、該当する行を表示
- replaceコマンド : ファイル内の文字列を別の文字列に置換
- createコマンド : 新規ファイルを作成
- deleteコマンド : ファイルを削除
- renameコマンド : ファイル名を変更
- mkdirコマンド : ディレクトリを作成
- rmdirコマンド : ディレクトリを削除
- lsコマンド : ディレクトリ内を一覧表示
- pwdコマンド : カレントディレクトリの絶対パスを表示
- cdコマンド : カレントディレクトリを変更
- seqrenameコマンド : ディレクトリ内のファイルを連番でリネーム
